# Food Delivery App

A Java Full Stack Food Delivery Web Application built using Java, JSP, Servlets, JDBC, and MySQL.

## Features

- User Registration
- User Login & Logout
- Restaurant Listing
- Search Restaurants
- View Restaurant Menu
- Add to Cart
- Checkout
- Order History
- Track Order
- User Profile
- Edit Profile
- Change Password

## Technologies Used

- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML5
- CSS3
- JavaScript
- Apache Tomcat

## How to Run

1. Clone this repository.
2. Import the project into Eclipse.
3. Configure Apache Tomcat.
4. Create the MySQL database.
5. Update the database connection details.
6. Run the project on Tomcat.
